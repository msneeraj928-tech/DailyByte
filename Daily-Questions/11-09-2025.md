## Problem Statement

Given a non-negative integer `n` in decimal form (base 10), convert it to its binary representation (base 2) and return the result as a string.

The binary representation should not have leading zeros, except when the number itself is 0.

## Example Implementation Ideas

### Approach 1: Division by 2 Method
- Handle the special case when n = 0.
- Repeatedly divide n by 2.
- Collect the remainder (0 or 1) at each step.
- Continue until n becomes 0.
- Reverse the collected remainders to get the binary representation.

### Approach 2: Bit Manipulation
- Find the position of the most significant bit.
- Iterate from that position down to 0.
- Check if each bit is set using bitwise AND operation.
- Build the binary string accordingly.

### Approach 3: Using Built-in Functions (Language Specific)
- Most languages have built-in functions like `bin()` in Python or `Integer.toBinaryString()` in Java.
- While using these is valid, try implementing the algorithm manually for learning.
